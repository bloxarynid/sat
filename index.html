Tentang Saya

Bloxaryn.Id

 Halo! Saya Bloxaryn.id (Dwika Hadi Wijaya). Cita-cita saya adalah menjadi seorang Front End Developer yang handal. Untuk mencapainya, saya saat ini aktif mempelajari dasar-dasar HTML dan CSS.

 Proses belajar saya dilakukan secara bertahap. DeepSeek menjadi teman diskusi yang membantu memperjelas pemahaman dan manfaat setiap kode, sementara TikTok serta media sosial memberi saya inspirasi template dan animasi kreatif untuk pengembangan visual.

 Saya sedang membangun fondasi yang kuat. Di HTML, saya mempelajari peran setiap tag, penggunaan atribut, dan seni menyusun serta mengelompokkan elemen dengan struktur yang rapi dan bermakna. Sementara di CSS, saya mendalami cara mengatur tata letak, menjembatani jarak antar elemen, memberikan gaya, menciptakan animasi sederhana, dan—yang krusial—memastikan setiap halaman web tetap responsif di berbagai perangkat.

 Rencana ke depan, setelah saya merasa cukup percaya diri dengan dasar-dasar ini, saya akan melangkah untuk mempelajari JavaScript untuk menghidupkan website dengan interaktivitas, lalu beralih ke Tailwind CSS untuk mempercepat dan memperdalam proses pengembangan antarmuka yang modern.

 Walaupun masih duduk di bangku SMP, saya tidak pernah goyah hanya karena melihat kemajuan orang lain, atau berbeda dengan impian mereka. Saya tetap fokus pada jalan saya sendiri. Di bawah ini, kalian bisa melihat semua proyek yang telah saya buat...
